# Duelist

Duelist is a web application that helps you organize your daily tasks with simplicity and style.

## Features

* Create and manage tasks
* Prioritize your tasks
* Organize your tasks
* Track your progress
* Secure authentication

## Technologies

* React
* Typescript
* Zustand
* React Router DOM
* Firebase
